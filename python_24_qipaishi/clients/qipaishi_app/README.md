# Qipaishi App

Flutter shell for the downloadable mobile and PC clients.

## Current Scope

- Shared Flutter app entry.
- Mobile and desktop layout split.
- API client placeholder.
- Session store placeholder.
- Basic widget test.

## First Run

Install Flutter SDK first, then run:

```bash
flutter pub get
flutter test
flutter run
```

To generate native platform folders after Flutter is installed:

```bash
flutter create . --project-name qipaishi_app --org com.qipaishi --platforms android,ios,windows,macos
```
